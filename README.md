# Personal Practicum Blog

A personal blog documenting my journey through the practicum program, starting from December 8, 2025.

## About

This blog serves as a weekly documentation of my experiences, learnings, and reflections during the practicum. It includes:

- **Profile**: Personal information, experience, skills, and achievements
- **Preparation**: Workshop activities and preparation for the practicum
- **Blog**: Weekly updates and reflections on the practicum journey

## Features

- Weekly blog posts documenting progress and learnings
- Image galleries showcasing workshop activities and experiences
- Personal profile with professional background
- Clean, modern design inspired by developer portfolios

## Content

The blog covers various aspects of the practicum experience, including:

- OJT application process
- Workshop activities (e.g., Office and Workplace Etiquette)
- Weekly reflections and learnings
- Progress updates
